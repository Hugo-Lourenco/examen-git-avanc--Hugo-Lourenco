la branche main est la branch de base du repo
la branche hotfix va permettre de réparer des fautes commises 
la branche dev va intégrer de nouvelles choses dans le main 
la branche features permet l'ajout de nouvelles fonctionnalité

j'ai utilisé git merge pour corriger le readme de dev car cela permet de prendre la correction du hotfix et de l'appliquer à dev sans conflit
